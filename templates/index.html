System overview

Hybrid risk intelligence for real-time transaction screening.

The system now follows a hybrid fraud-detection pipeline: regional behavior learning from the Indian dataset, global card-pattern learning from the European dataset, and a fusion layer that combines both views into one decision.

A unified scoring engine converts those signals into one risk score, one risk level, and one dynamic explanation stream for every transaction profile submitted through the interface.
